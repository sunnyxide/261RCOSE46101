## S01 — 헬스장 12개월 결제: 락인과 후회

**인물**: 김○○, 32세, 직장인 여성. 월 소득 280만 원. 슬슬 살이 찐다는 스트레스.

**서비스 결정**: 헬스장 12개월 회원권 + PT 30회 패키지. 12개월 회원권 60만 원, PT 30회 150만 원(1회 5만 원), 총 210만 원 선결제. "12개월 결제 시 2개월 무료" 조건.

**결정 트리거**: 동료가 "나 3개월 다녔는데 5kg 뺐어" → 30대 초반 여성의 자기관리 사회적 기대 + Naver 블로그 "30대 다이어트 성공기" 노출.

**대안 고려**:
1. 홈트레이닝 (유튜브 "땅끄부부" 팔로잉) — 0원, 하지만 3번 시도하고 포기 경험
2. 근처 헬스장 1개월 결제 (월 5만5천 원) — 부담 없지만 "비효율적" 인식
3. PT 포함 12개월 패키지 — 초기 비용 높지만 "투자" 프레이밍에 끌림

**정보 탐색**:
- Naver 카페 "○○동 헬스장 후기" — 실제 회원 후기, PT 강사 평가
- Naver 블로그 "PT 등록 후기 3개월" — 수십 개 리뷰 탐색
- 헬스장 상담 시 "체성분 분석" 무료 제공 → 전문가 프레이밍
- 카카오톡 PT 상담 — 상담사가 "3개월만에 확 달라진다" 사례 공유

**구매 흐름**:
1. 헬스장 무료 체험 1회 (체성분 분석 포함)
2. 상담사가 "12개월+PT 30회 세트가 가장 효율적" 제안
3. "지금 등록하면 30회 추가 서비스" 한정 할인 → 손실회피 자극
4. 신용카드 일시불 210만 원 결제
5. 1-2개월은 주 3회 출석

**구매 후**:
- 3개월차: 업무 야근 증가, 주 1회로 감소
- 5개월차: PT 횟수 소진, 헬스장만 다니기 시작 → "PT 없으면 의미 없나" 자괴감
- 8개월차: 출석 거의 안 함. 남은 기간에 대한 위약금 문의 → "잔여 기간 50% 환불" 안내
- 10개월차: 50% 환불 거부, "다시 다녀야지" 마음 → 실제 출석 0회
- 최종: 210만 원 중 실사용 PT 14회(70만 원), 실출석 약 50회. 실효 단가 약 2.4만 원/회

**BAS 메트릭**:
- **선결제 락인效应**: 12개월 결제 → 취소 시 위약금 50% → 3개월 후 후회 시작
- **선행 투자 오류 (Sunk Cost)**: "210만 원이나 냈으니 더 다녀야지" → 실제 행동 변화 없음
- **현상유지 편향**: 출석 안 하는 상태가 "新常态"이 됨

**한국적 맥락**:
- 한국 헬스장 결제 구조: 연간 선결제가 기본이며, 월 결제 옵션은 30-50% 비쌈. 소비자를 장기 락인으로 유도하는 구조적 설계
- KCA(한국소비자원) 서비스 분쟁 통계: 헬스장 분쟁은 "계약 해지·환불"이 전체의 70% 이상(2023년 기준 약 4,200건). 헬스장 폐업 시 환불 불가 사례 다수
- 한국 20-30대 여성의 헬스장 등록률은 높으나 6개월 내 중도 해지 비율 추정 60% 이상

**데이터 신호**:
| 신호 | 값 | 출처 |
|---|---|---|
| PT 평균 단가 | 5-8만 원/회 | Naver 헬스장 PT 후기 수집(2024) |
| 12개월 회원권 평균 | 50-80만 원 | Naver 지역 카페 시세 비교 |
| KCA 헬스장 분쟁 건수 | ~4,200건/년 (추정) | KCA 서비스 분쟁 통계(2023) |
| 중도 해지 시 위약금 비율 | 잔여 기간 30-50% | 헬스장 표준약관(공정위) |

## S02 — 영어유치원 입학: 단톡방 압력과 사교육비

**인물**: 이○○, 35세, 전업주부. 남편 월 소득 450만 원. 딸(5세) 1명. 아파트 단지 내 유치원·어린이집 정보에 민감.

**서비스 결정**: 영어유치원 등록. 월 120만 원(급식비 포함), 입학금 50만 원, 교재비 30만 원. 연간 약 1,540만 원. 원어민 교사 비율 80% 이상.

**결정 트리거**: 같은 반 엄마 5명 중 3명이 "영어유치원 보냈다"고 단톡방(카카오톡 단체 채팅)에 공유. "요즘은 다 보내" 발언. 남편은 "그 돈이면 적금 드는 게 낫다" 반대.

**대안 고려**:
1. 국공립 유치원 + 주 2회 영어 학원 — 월 30만 원. "형평성" 논리로 합리적
2. 병설유치원(방과후 영어 프로그램) — 월 15만 원. "학교 적응" 장점
3. 영어유치원 — 월 120만 원. "글로벌 인재" 프레이밍

**정보 탐색**:
- 카카오톡 단톡방 "○○아파트 유치원 정보" — 30명 규모, 실시간 경험 공유
- 맘스홀릭 카페 "영어유치원 비교 후기" — 수백 개 글, 지역별 순위
- Naver 블로그 "영어유치원 솔직 후기 1년" — 장단점 균형 후기
- 한국교육개발원 사교육 실태조사(2023): 학령 전 사교육 참여율 약 41.6%, 월 평균 사교육비 약 40.5만 원
- 유치원 설명회 참석 — 원어민 교사 시범 수업, 시설 견학

**구매 흐름**:
1. 단톡방에서 "○○영어유치원 원장이 좋다" 후기 등장
2. 남편 설득 시작: "다른 애들 다 보내는데 우리만 안 보내면 뒤처진다"
3. 설명회 참석 → 아이가 원어민 선생님과 잘 노는 모습 확인
4. 입학금 50만 원 + 첫 달 120만 원 결제
5. 대기자 명단에 있다가 3월 입학 확정

**구매 후**:
- 1개월: 아이 적응期, 울면서 등원. 엄마 불안 시작
- 3개월: 아이 영어 노출 증가, "apple" 말하기 시작 → 엄마 안도
- 6개월: 단톡방에서 "다른 애는 이미 영어로 문장 말한다" 비교 → 조급함
- 12개월: 1년간 1,540만 원 지출. 아이는 영어에 흥미 보이지만 "초등학교 가면 잊는다"는 주변 말에 불안
- 재등록 여부 결정 시점 — "1년 더 보내야 효과 있다"는 학원 측 + 단톡방 분위기 → 재등록

**BAS 메트릭**:
- **사회적 동조 (Conformity)**: 단톡방 내 "다 보내" 발언 → 개인 판단 약화
- **선행 투자 오류**: 1,540만 원 지출 후 "중간에 그만두면 아깝다" → 재등록
- **비대칭 정보**: 학원 측 "효과" 주장 vs 객관적 학습 성과 측정 불가
- **FOMO**: "우리 아이만 안 보내면 뒤처진다" 공포

**한국적 맥락**:
- 한국 사교육비는 OECD 최고 수준. 통계청 가계동향조사(2023): 가구당 월 평균 사교육비 약 40.5만 원(학령 전), 초·중·고 합산 약 44.3만 원
- 단톡방(카카오톡)은 한국 아파트 단지 학부모의 주요 정보 채널. 정보 비대칭 → 단톡방 내 "사실"이 곧 진실이 되는 구조
- 한국교육개발원 사교육 실태조사(2023): 학령 전 사교육 참여율 약 41.6%. 영어유치원 월 평균 100-150만 원

**데이터 신호**:
| 신호 | 값 | 출처 |
|---|---|---|
| 학령 전 사교육 참여율 | 약 41.6% | 한국교육개발원 사교육 실태조사(2023) |
| 월 평균 사교육비(학령 전) | 약 40.5만 원 | 통계청 가계동향조사(2023) |
| 영어유치원 월 평균 비용 | 100-150만 원 | Naver 맘카페 시세 조사(2024) |
| 재등록률(영어유치원) | 추정 70-80% | 학원 업계 관행(확인 필요) |

## S03 — 미용실 단골 vs 신규 시도: 전환 비용과 후기 신뢰

**인물**: 박○○, 27세, 여성. 직장인. 3년간 같은 미용실(강남역 인근) 다님. 기본 커트 3만5천 원, 전체 염색 8만5천 원.

**서비스 결정**: 새 미용실 시도 여부. 친구 추천(홍대 "○○살롱") + 인스타그램(@○○hair) 발견. 커트 2만5천 원, 염색 6만5천 원. 20-30% 저렴.

**결정 트리거**: 1) 친구가 "홍대에서 염색했는데 훨씬 예쁘고 싸" 후기 공유, 2) 인스타그램 릴스에서 트렌디한 스타일 발견, 3) 현재 미용실 원장이 최근 커트 실수 → 불만 누적.

**대안 고려**:
1. 현재 미용실 유지 — 단골 할인 10%, 원장이 내 머리 특성 파악. 감정적 유대
2. 친구 추천 홍대 살롱 — 저렴 + 친구가 보증. 단, 1시간 거리
3. 인스타그램 발견 강남 살롱 — 트렌디, 단 미용사 바뀔 수 있음

**정보 탐색**:
- 인스타그램 해시태그 "#홍대미용실 #염색맛집" — 사진 기반 후기, 비포/애프터
- 네이버 블로그 "홍대 ○○살롱 솔직 후기" — 텍스트 후기, 가격 정보
- 인스타그램 릴스/유튜브 쇼츠 — 미용사 시술 영상, 스타일 확인
- 강남역 미용실 네이버 평점 — 4.7점(327건 리뷰). "단골 많다" 평가
- 카카오톡 친구에게 "실제로 어땠어?" 직접 문의

**구매 흐름**:
1. 새 미용실 카카오톡 예약 시도 → 2주 후 예약 가능
2. 예약 전 "상담만" 방문 → 미용사가 내 머리 상태 보고 스타일 제안
3. 상담이 만족스러워 염색 예약 확정
4. 시술 당일: 기대와 차이 → 미용사가 제안한 색이 내 피부톤과 안 맞는다고 느낌
5. 결과물은 "괜찮은" 수준. 하지만 "내 단골이었으면 더 잘 맞췄을 텐데" 아쉬움

**구매 후**:
- 1주일: 염색 결과물에 점차 적응. 사진 찍어 SNS 업로드
- 1개월: 뿌리 자라남 → 단골 미용실 갈지 새 미용실 갈지 고민
- 최종: 단골 미용실 복귀. 새 미용실은 "한 번 가볼 만했지만" 결론
- 전환 시도 비용: 시간 3시간(왕복 1시간 + 시술 2시간), 감정적 불안정

**BAS 메트릭**:
- **전환 비용 (Switching Cost)**: 3년간 축적된 단골 관계 + 미용사가 내 머리 특성 파악 → 재전환 시도 비용 큼
- **사회적 증거 (Social Proof)**: 친구 직접 후기 > 인스타그램 후기. "아는 사람" 정보가 신뢰도 최상위
- **현상유지 편향**: 결과가 "괜찮아도" 단골로 복귀하는 경향
- **탐색 비용**: 새 미용실 상담 + 예약 + 이동 → 3시간 이상의 시간 투자

**한국적 맥락**:
- 한국 미용실 시장: 약 15만 개(2024년). 평균 영업기간 약 3.2년 → 단골 확보가 생존 조건
- 미용실 단골 시스템: 포인트 적립, 커트 할인, 스타일 기록. 전환 시 "잃는 것" 큼
- 한국 소비자 미용실 선택 시 "가격"보다 "후기"와 "추천" 중시. 인스타그램/네이버 블로그 후기가 의사결정의 1순위(2024년 미용 업계 트렌드)

**데이터 신호**:
| 신호 | 값 | 출처 |
|---|---|---|
| 한국 미용실 수 | 약 15만 개 | 소상공인시장진흥공단(2024) |
| 미용실 평균 염색 가격 | 6-10만 원 | Naver 미용실 가격 비교(2024) |
| 단골 유지율(3년 이상) | 추정 40-50% | 미용 업계 관행 |
| 미용실 선택 시 "후기/추천" 비중 | 추정 60% 이상 | 미용 업계 소비자 조사(2023) |

## S04 — 통신 3사 결합상품: 가족 단위 교섭과 약정 락인

**인물**: 정○○(45세, 남편), 최○○(42세, 아내), 자녀 2명(중1, 초5). 4인 가구. 현재 SKT 모바일 4회선 + LG U+ 인터넷/TV 사용. 월 통신비 총 약 26만 원.

**서비스 결정**: SKT "온가족할인" 결합상품 전환 검토. 모바일 4회선 + 인터넷 + IPTV + 넷플릭스 번들. 약정 36개월. 월 예상 18만5천 원(약 7만5천 원 절감). 단, LG U+ 인터넷 해지 위약금 약 15만 원.

**결정 트리거**: 1) SKT 대리점에서 "온가족할인" 문자 수신 — "지금 전환하면 넷플릭스 1년 무료", 2) 남편 직장 동료가 "SKT 결합 쓰면 훨씬 싸다" 경험 공유, 3) LG U+ IPTV 앱이 느림 → 불만 누적.

**대안 고려**:
1. 현상 유지 (SKT 모바일 + LG U+ 인터넷) — 월 26만 원. LG U+ IPTV 불만 있으나 익숙함
2. SKT 결합 전환 — 월 18만5천 원. 36개월 약정, LG U+ 위약금 15만 원 부담
3. KT 결합 검토 — 월 20만 원. 단, 자녀 학교에서 "SKT가 잘 터진다"는 평판

**정보 탐색**:
- SKT 공식 앱 "T world" — 결합상품 시뮬레이션, 월 예상 요금 계산
- 네이버 카페 "뽐뿌" — 통신사 결합 비교 글, 실제 후기 수백 건

2. 주말 1회 대여(4시간) → 실제 경험. 차량 상태 확인
3. 쏘카패스 가입(월 9만9천 원) → 4회 50% 할인 혜택 확인
4. 월 4회 주말 대여 패턴 확립

**구매 후**:
- 1개월: 주말마다 대여 → 여자친구와 드라이브, 부모님 댁 방문
- 3개월: 월 대여 횟수 6회로 증가 → 추가 대여비 발생(패스 혜택 초과)

- 12개월: 연간 쏘카 비용 약 216만 원. 자차 보유 비용(연 420만 원)의 절반
- 최종: 쏘카패스 유지 결정. "자차 사면 주차비가 더 문제" → 한국 주차 환경이 카셰어링 지속 요인

**BAS 메트릭**:
- **자차 대체 결정**: 자차 구매(고정비 35만/월) vs 카셰어링(가변비 18만/월) → 비용 비교 기반 합리적 결정
- **구독 전환**: 개별 대여(24만/월) → 패스 구독(18만/월) → "구독=할인" 인식
- **행동 변화**: 자차 보유 시 "편의성" 프레이밍 vs 카셰어링 시 "비용 절감" 프레이밍

**한국적 맥락**:

## S09 — 부동산 전세 vs 매매: 전세가 상승과 "벼락거지" 불안

**인물**: 오○○(30세, 남편, 직장인), 장○○(29세, 아내, 프리랜서). 결혼 1년차. 현재 오피스텔 전세 8천만 원. 내년 전세 만기.

**서비스 결정**: 전세 갱신 vs 매매 전환(소형 아파트). 전세 1억2천(4천 인상) vs 매매 2억5천(대출 1억8천). 부모 지원: 없음(양가 모두 어려움).

**결정 트리거**: 1) 집주인이 전세 4천 인상 통보 → "이 돈이면 대출 이자와 비슷하다" 계산, 2) 네이버 부동산 카페에서 "전세가 폭등" 글 쇄도 → 불안, 3) 직장 동료가 "지금 아니면 못 산다" 발언.

**대안 고려**:
1. 전세 갱신 — 1억2천. 2년 더 거주. 매매 자금 모으기
2. 소형 아파트 매매 — 2억5천. 대출 1억8천(연 4.5%), 월 상환 약 92만 원
3. 경기도 외곽 전세 — 1억. 출퇴근 1시간30분

**정보 탐색**:
- 호갱노노 — 전세가 상승 추이 확인, 단지별 전세가율
- 직방·다방 — 매물 탐색, 가격 비교
- 네이버 부동산 카페 — "벼락거지" 논란, 전세 vs 매매 분석 글
- 부모님께 상담 → "우리는 도와줄 여력이 없다" → 독립적 결정 강요

**구매 흐름**:
1. 부동산 앱으로 소형 아파트 5곳 리스트업
2. 주말 방문(매물 실물 확인)
3. 대출 상담(은행) → 1억8천 대출 가능 확인
4. 부부 논의: "월 대출 상환 92만 원 vs 전세 인상분 4천/2년"
5. 최종: 소형 아파트 매매 결정. 대출 1억8천 + 전세금 반환(8천) → 자금 조달

**구매 후**:
- 계약 후: 중도금 납부, 대출 실행
- 입주 후: 월 대출 상환 92만 원 + 관리비 12만 원
- 3개월: 부동산 시세 정체 → "오를까?" 불안
- 6개월: 금리 인상 → 변동금리 이자 증가. "고정금리 할걸"
- 12개월: 시세 하락 → "지금이 아니라 1년 후에 살걸" 후회
- 최종: "그래도 내 집은 내 집" 위안. 하지만 월 고정비 부담으로 소비 축소

**BAS 메트릭**:
- **벼락거지 불안**: 전세가 상승 → "지금 아니면 못 산다" 시간 압박
- **부모 지원 없음**: 양가 모두 지원 불가 → 부부 단독 결정. 한국에서 드문 경우(50% 이상 부모 지원)
- **대출 부담**: 월 상환 92만 원(소득 대비 30% 이상) → 소비 축소
- **전세 vs 매매 비용 역전**: 전세 인상분 ≈ 대출 이자 → 매매 전환 유인

**한국적 맥락**:
- 전세 제도 특성: 전세금 반환 불안정(전세사기 등) → 매매 전환 유인
- 부모 지원 불가 비율: 한국 20-30대 첫 집 마련 시 부모 미지원 비율 추정 약 40-50%
- "벼락거지" 담론: 부동산 가격 상승 → 자산 미보유자의 상대적 박탈감. 한국 특유의 부동산 중심 자산관

**데이터 신호**:
| 신호 | 값 | 출처 |
|---|---|---|
| 전세가 상승률(수도권, 2023-2024) | 약 5-10% | KB부동산 시세 |
| 소형 아파트 평균 매매가(수도권) | 2-3억 | 호갱노노 실거래가 |
| 대출 이자율(2024) | 약 4-5% | 한국은행 기준금리 + 가산 |

## S10 — 프리미엄 구독 서비스: 커피·와인·꽃 — 자아 정체성과 죄책감

**인물**: 조○○, 33세, 여성. 직장인. 연봉 4,200만 원. 무자녀. "나를 위한 소비" 중시.

**서비스 결정**: 프리미엄 구독 서비스 3종 가입. 커피 구독(커피앳홈 월 3만5천 원), 와인 구독(빈토 월 5만5천 원), 꽃 구독(꽃달 월 2만5천 원). 월 합계 11만5천 원.

**결정 트리거**: 1) 인스타그램에서 "홈카페" 라이프스타일 콘텐츠 노출 → "나도 저렇게 살고 싶다", 2) 직장 동료가 "커피 구독 시작했는데 정말 좋다" 경험 공유, 3) 월급 인상(5%) → "나를 위한 투자" 정당화.

**대안 고려**:
1. 구독 없이 개별 구매 — 스타벅스(월 5만 원) + 편의점 와인(월 2만 원) + 꽃(월 1만 원). 합계 8만 원
2. 커피 구독만 — 월 3만5천 원. 나머지는 개별 구매
3. 3종 구독 — 월 11만5천 원. "프리미엄" 경험 + 정기 배송 편의

**정보 탐색**:
- 인스타그램 해시태그 "#커피구독 #와인구독 #꽃구독" — 라이프스타일 콘텐츠, 언박싱 영상
- 네이버 블로그 "커피앳홈 후기" — 원두 맛 평가, 배송 품질
- 유튜브 "프리미엄 구독 서비스 비교" — 3종 비교 리뷰
- 카드사 앱 — 구독 서비스 할인 프로모션 확인(신한카드 10% 할인)

**구매 흐름**:
1. 인스타그램에서 "커피앳홈" 광고 노출 → 클릭 → 상품 페이지
2. 첫 달 50% 할인 쿠폰 → "한 달만试试"
3. 커피 구독 시작 → 원두 품질 만족
4. "와인도, 꽃도 해볼까" → 3종 구독 확대
5. 언박싱 사진 인스타그램 업로드 → "프리미엄 라이프스타일" 자기표현

**구매 후**:
- 1개월: 매주 배송되는 커피·와인·꽃 → "내가 나를 위해 투자하는 느낌" 만족
- 3개월: 월 11만5천 원 지출 확인 → "좀 과하지 않나" 죄책감 시작
- 6개월: 와인 2병 중 1병 미개봉 → "다 못 마시는데" 낭비 인식
- 9개월: 꽃 구독 해지 → "花는 시들면 버리는 거라 돈 낭비"
- 12개월: 커피 구독 유지, 와인 구독 해지 → 월 3만5천 원으로 축소

**BAS 메트릭**:
- **자아 정체성 소비**: "나를 위한 투자" + 인스타그램 자기표현 → 구독=정체성

- **SNS 영향**: 인스타그램 라이프스타일 콘텐츠 → 구매 유도 → 실제 생활과 괴리
